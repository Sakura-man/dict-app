# 架空言語翻訳アプリ

日本語を架空言語（アルファベット＋ギリシャ文字）に翻訳するWebアプリです。  
ブラウザから簡単に使用できます。

---

・起動方法（ローカル）
①フォルダに移動
```bash
cd /d E:\dict_app
② 仮想環境を起動
venv\Scripts\activate
③ サーバー起動
python -m uvicorn app:app --reload
④ ブラウザで開く
http://127.0.0.1:8000

・更新方法（重要）
コードや辞書を変更した場合：
git add .
git commit -m "update"
git push
自動でRenderに反映されます（数十秒〜数分）

・辞書の仕組み
dictionary.json に単語を定義します。
基本形式
"亜語": ["日本語1", "日本語2"]
例
"i": ["わたし", "自分"],
"aρρre": ["りんご"]

・単語の追加方法
① dictionary.json を開く
② 新しい単語を追加する
"yoτα": ["あなた"]
③ 保存
④ pushする
git add .
git commit -m "add word"
git push

・注意（重要）
カンマ忘れ

"りんご": ["aρρre"]
"みかん": ["orange"] ← ❌

正しくは

"りんご": ["aρρre"],
"みかん": ["orange"]

・翻訳仕様
日本語はスペース不要
例：巨大な店 → hoμge sζope
長い単語から優先して翻訳
未登録の単語はそのまま表示される

・対応文字
アルファベット
ギリシャ文字（Noto Sansフォント使用）
スマホ対応
フォント：Noto Sans
一部の特殊文字は機種によって表示が崩れる可能性あり

・使用技術
Python
FastAPI
Uvicorn
Render（デプロイ）
GitHub（管理）
